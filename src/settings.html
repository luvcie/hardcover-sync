important: your api token is personal and should never be shared. it provides full access to your hardcover account.
get your token from hardcover.app/account/api

how to get your api token

  1. go to hardcover.app/account/api
  2. log in to your hardcover account
  3. copy the bearer token from the page
  4. paste it in the field above
  5. click "save token"
note: tokens expire after 1 year. you'll need to regenerate it when it expires.

keyboard shortcuts

disable this if keyboard shortcuts conflict with other extensions

available shortcuts:

  • Ctrl+Shift+. - open extension popup
  • Ctrl+Shift+, - toggle floating window
  • Ctrl+Shift+F - toggle floating window (alternative)
  • Arrow Up - increment page number
  • Arrow Down - decrement page number
  • Enter - submit/update progress
note: arrow keys and enter only work when the page input is focused

about this extension

this extension uses hardcover's api to track your reading progress for pdfs. since hardcover's api is designed for backend use, please be aware:

  • your api token is stored locally in your browser
  • never share your token or browser profile
  • this is for personal use only
